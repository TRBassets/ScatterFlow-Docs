Installation
  1. Download the ScatterFlow.zip file
  2. In Blender 4.4, open Edit ▸ Preferences ▸ Extensions and select Install from disk.

Add ScatterFlow assets in the Asset Library

  1. Navigate to Edit ▸ Preferences ▸ File Paths ▸ Asset Libraries.
  2. Select the + icon to create a new library entry.
  3. Choose the directory containing ScatterFlow Assets.blend
  4. Click Save Preferences to store the path.
Quick Start

Drop objects

  1. Navigate to Tools > Simulation Tools.
  2. Select the floor (or any collision surface) and click Set Passive.
  3. Select your objects and click Add Physics.
  4. Press Space to play the simulation.

Spawning objects

  1. Click Add Tool to add the spawning sphere.
  2. Select the desired collection from Source Collection.
  3. Navigate to Spawning > Quick Spawn.
  4. Press Spawn Instances.
  5. Ensure a passive floor or objects exist so the instances collide with something.
Features
Spawning
Quick Spawn

Generate instances from a spawn sphere with physics already applied.

  • Count – number of objects generated per click.
  • Scale Min/Max – size range.
  • ∞ Spawn – continuous spawning during playback.
    Rate – objects per second.
  • Random Rotation – if on, spawns objects with randomised orientation; if off, uses applied rotation.
  • Randomize Selected – deletes the spawned objects and generates only the selected objects.
  • Randomize All – deletes the spawned objects and generates a new set.
Surface Scatter

Scatter objects across a surface.

  • Target Mesh – surfaces that objects will spawn on.
  • Count – number of objects scattered.
  • Random Rot – randomised orientation.
  • Skip Rigid Bodies – spawn objects without physics applied.
  • Randomize Selected – deletes the spawned objects and generates only the selected objects.
  • Randomize All – deletes the spawned objects and generates a new set.
Array Spawn

Create structured grids of instances for organised layouts.

  • Count X/Y/Z – how many object rows are spawned.
  • Gap X/Y/Z – spacing between objects.
  • Scale Min/Max – size variation.

Tools
Manipulation Tool (Force‑Field)

Interactively pick up, release, drag and throw rigid‑body objects using a configurable force field.

  • Dynamic Pickup – toggles field strength for grabbing or releasing objects.
  • Drag – Sets the strength value to -500 for dragging objects .
  • Strength – Controls how strong the attraction force is.
  • Quick tip: press U – switch between pickup and drop modes.
Simulation Tools

Tool for managing or adjusting your simulated objects

  • Add Physics – Add physics to selected objects.
  • Set Passive – Turns selected objects into a static collision object.
  • Duplicate – Duplicates the selected objects.
  • Move – enable Objects to be moved. (Must pause before enabling).
  • Freeze – converts selected active physics objects to passive. Useful for freezing selected objects in place when finished.
  • Remove Physics – Completely removes physics from selected objects.
Quick Array Grid

An alternative tool to array spawning. This tool allows you to quickly array a single object.

  • Count X/Y/Z – how many object rows are spawned.
  • Gap X/Y/Z – spacing between objects.
  • Note: Quick array objects do not include physics. Select and press the add physics button to add physics.
Cloth Tools

Tools for managing your cloth simulation.

  • Add Cloth – Adds a cloth simulation to the selected objects.
  • Add Collisions – create colliders for cloth objects.
  • Remove Collisions – remove colliders from the selected objects.
  • Apply Cloth – Applies the cloth simulation.
Advanced
Advanced Settings
  • Split Impulse – Reduces physics velocity applied to objects. Off = more explosive force / On = less force.
  • Friction/Bounce – Adjusts the level of friction and bounce of objects.
  • Collision Shape – adjusts the collision type for each object. Convex hull = faster but less accurate, Mesh = slower but more accurate.
  • Set as Default – save current parameters as defaults for newly spawned objects.
  • Origin Tools – quickly switch the origin to center or base of objects; helpful for scattering vs. simulation tasks.
Common Issues
  • Objects pass through floor – ensure the floor object is set to Passive and that its Collision Shape is appropriate (e.g. BOX for cubes, MESH for complex geometry).
  • Pile explodes on impact – lower Bounce (phystool_adjust_bounce) and enable Split Impulse.
  • Viewport performance is reduced – activate Link Mesh Data for instancing or Skip Rigid Bodies for static decoration.
FAQ & Tips

Q: How do I replay the simulation?
A: Scrub the timeline or press Space—Blender automatically caches the rigid‑body simulation.

Q: Can I use custom force curves?
A: Convert the spawn sphere to a force‑field (Swap Tool) and adjust Strength and Flow.
